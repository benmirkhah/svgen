# svgen

<b>Generates SVG with random elements</b>

I love SVG files, specially for page backgrounds, <a href="https://github.benmirkhah.com/svgen/">see it in action</a>.

This is a work in progress, so far just circles, squares, and puffy clouds, more shapes will be added soon.

Granular config object allows fine adjustment of all parameters used to generate the shapes.
